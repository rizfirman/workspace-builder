## Workspace Builder – 3D Configurator

This project is an interactive 3D workspace configurator that allows users to visually design their desk setup by selecting desks, chairs, and accessories in real-time.

### Approach

I focused on building a spatially-aware layout system where the desk acts as the main anchor. All objects (chair, monitors, accessories) are positioned relative to it using bounding box calculations to ensure realistic placement.

### Tech Stack

- Next.js (App Router)
- React Three Fiber (Three.js)
- Zustand (state management)
- Tailwind CSS

### Improvements

With more time, I would add drag-and-drop with snapping, smoother animations (GSAP), better asset standardization, and a more advanced lighting and camera system for a more immersive experience.
